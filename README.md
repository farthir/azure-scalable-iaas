# azure-scalable-iaas
ARM templates and Chef cookbooks for deployment of a simple, scalable IaaS based web application.

## Requirements
Tested only with:

1. Microsoft Azure
2. Ubuntu 14.04.4-LTS
3. nginx 1.4.6
4. Apache Tomcat 8.5.3
